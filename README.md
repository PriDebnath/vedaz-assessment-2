# Basic chat application.

A real-time chat application using  React for the frontend 
and Node.js + Express + Socket.io for the backend 
made by Pritam Debnath

## Feature
- User authentication flow
- Send messages other user
- Fetch chat history
- Real-Time Communication 
- Online/offline user status.

# Project setup instructions.
## Steps to run the frontend.
```
cd vedaz-assessment-2
``` 
```
pnpm install
``` 
```
npm run corepack:enable
``` 
```
cd frontend
``` 
```
npm run dev
``` 
```
http://localhost:5173/
```

## Steps to run the backend.
```
cd vedaz-assessment-2
``` 
```
pnpm install
``` 
```
npm run corepack:enable
``` 
```
cd backend
``` 
```
docker compose up
```
--
```
cd backend
``` 
```
npm run dev
```
## Environment variables.

#### create a `.env` file with the content of `.env.example` file
